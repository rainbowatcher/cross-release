# Cross Release

> a release tool that support multi programming language

## Feature

- Support Java/JavaScript/Rust 
  > current support `package.json`/`pom.xml`/`Cargo.toml`
- Prompt based cli operation
- Monorepo support
- Easy to use

| ------------------------------------------------------------------------------------------------------------------ | ---------------------------------- |
